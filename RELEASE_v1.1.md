# 🌊 Prismarine v1.1 - The Custom Era

今回のアップデートでは、カスタムサーバーの管理機能を大幅に強化し、動作の安定性を極限まで高めました。

---

## 🚀 主要な新機能

### 1. カスタムJARのドラッグ＆ドロップ(D&D)

任意の `.jar` ファイルをアプリケーションにドロップするだけで、即座にカスタムサーバーとして登録・作成。Windowsの複雑なディレクトリパスも自動でスマートに処理します。

> [!TIP]
> ドロップ後に表示される設定画面で、サーバー名やバージョン表示を自由に変更できます。「Manual」ではなく、自分好みのラベル（例: `Minecraft 1.21.1`）を設定しましょう！

### 2. Javaバージョンの自由選択 ＆ 手動構成

サーバー詳細画面から、使用するJava（8, 17, 21, 25など）を**数値を入力するだけで自由に変更可能**。

> [!IMPORTANT]
> 適切なJavaバージョンを選択しないと起動に失敗する場合があります：
> * **1.20.5+**: Java 21 (または 25) 推奨
> * **1.17 - 1.20.4**: Java 17 推奨
> * **1.16.5以下**: Java 8 推奨

### 3. PID監視によるリアルタイムなステータス同期

OSのプロセスリスト（PID）を2秒ごとに直接チェック。サーバーが不意に落ちたり、外部から強制停止されたりした場合でも、即座にUIを「Stopped」状態へ同期させます。

> [!NOTE]
> これにより「停止ボタンを押したのに表示が変わらない」といった同期ズレが解消され、非常にレスポンスの良い管理体験が実現しました。

### 4. デフォルトJava 21 への引き上げ

現代のMinecraftサーバーの主流に合わせ、カスタムサーバーなどのバージョン不明時のデフォルトJavaをJava 21へ変更。最新のJARなら「ドロップして即起動」が可能です。

## 🛠️ その他の改善箇所

- **ポート汚染の完全防止**: アプリ終了時に、管理下の全プロセスを確実にクリーンアップします。
- **UIステータス同期の高速化**: 停止コマンド送信後、プロセスが消えた瞬間にUIも切り替わるようになりました。
- **ビルドの安定性向上**: 最新のTauri v2 + Rust環境における依存関係の不備をすべて一掃しました。
